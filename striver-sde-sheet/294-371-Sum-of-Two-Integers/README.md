# Sum of Two Integers (Medium)

**Problem ID:** 371  
**Problem Number:** 294  
**Link:** https://leetcode.com/problems/sum-of-two-integers

## Problem Statement

Given two integers a and b, return the sum of the two integers without using the operators + and -.

&nbsp;
Example 1:
Input: a = 1, b = 2
Output: 3
Example 2:
Input: a = 2, b = 3
Output: 5

&nbsp;
Constraints:


	-1000 &lt;= a, b &lt;= 1000
